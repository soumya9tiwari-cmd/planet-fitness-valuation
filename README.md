# Planet Fitness Valuation

## Overview

Developed a comprehensive valuation model for Planet Fitness using a Discounted Cash Flow (DCF) approach supported by financial forecasting, comparable company analysis, working capital projections, and sensitivity analysis. The model estimates enterprise value, implied equity value, and intrinsic share price under multiple operating scenarios.

---

## Project Objectives

- Forecast Planet Fitness' financial performance
- Estimate enterprise and equity value using DCF
- Calculate an appropriate discount rate (WACC)
- Analyze valuation sensitivity to key assumptions
- Compare valuation outcomes across different scenarios

---

## Model Components

### Financial Forecasting
- Revenue projections
- Cost of Goods Sold (COGS)
- SG&A projections
- EBITDA and EBIT forecasts
- Capital expenditure projections
- Depreciation & Amortization
- Free Cash Flow (FCF) projections

### Valuation
- Discounted Cash Flow (DCF)
- Enterprise Value
- Equity Value
- Implied Share Price
- Terminal Value
- Mid-Year Convention

### Supporting Analysis
- WACC estimation
- Comparable company analysis
- Working capital projections
- Scenario analysis (Base, Upside, Downside)
- Sensitivity analysis
